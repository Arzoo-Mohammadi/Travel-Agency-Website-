# Travel-Agency
 # Travel Agency Website

A modern and responsive travel agency website built with Tailwind CSS,HTML , JS with a modem animation to showcase travel destinations, tour packages, and tourism services.

## Project Overview

This project is a travel agency website designed to provide users with an easy way to explore travel destinations and discover tour packages. The website focuses on clean design, responsiveness, and a smooth user experience.

## Features

* Modern and responsive design
* Modern Animations
* Travel destination showcase
* Tour packages section
* User-friendly navigation
* Mobile-friendly layout

## Technologies Used

* HTML
* Tailwind CSS
* JavaScript
